# Crumble & Co. — Artisan Bakery Website

A responsive single-page marketing website for an artisan bakery.

## Live Demo
[View on Vercel](https://crumble-co.vercel.app)

## Built With
- HTML5
- CSS3 (custom properties, Grid, Flexbox, keyframe animations)
- Vanilla JavaScript

## Features
- Animated hero section with Unsplash photography
- Interactive menu with category filtering
- Scroll-reveal animations
- Fully responsive down to mobile
- Zero dependencies — no frameworks or libraries
